1. git init // 初始化一个git仓库 2.git add /. // 添加到本地储存 3.git commit -m 'xxx' //把本地存储存储在仓库，-m后面输入的是本次提交到仓库的说明 4.git status //查看的状态 5.git diff // 查看更改了那些内容 6.版本回退 git log [--pretty=oneline]//显示从最近到最远的提交日志 HEAD表示当前版本，上一个版本就是HEAD^，上上一个版本就是HEAD^^ git reset --hard "HEAD^" // 返回上一个版本 或者 git reset --hard commit_id(版本号) git reflog // 查看命令历史 7.丢弃工作区修改 git checkout -- 这是保存了还没有操作git add .... git reset HEAD 写了一些胡话，还git add到暂存区了： 8.删除文件 eg：把没用的del.html文件删了 现在你有两个选择，一是确实要从本地存储中删除该文件，那就用命令git rm删掉，并且git commit： git rm del.html git commit -m "remove del.html" 另一种情况是删错了，因为版本库里还有呢，所以可以很轻松地把误删的文件恢复到最新版本： git checkout -- del.html 9.git 和 github的远程操作 1) 在github上创建一个新的远程仓库 2）将本地仓库和远程仓库相关联 git remote add origin https://github.com/wuyanping/gitLx.git 3）git push -u origin master 将本地仓库push到远程仓库上 10.从远程上克隆 git clone https://github.com/wuyanping/gitskills2.git 11.分支 1)创建分支 git checkout -b dev //创建dev分支，然后切换到dev分支 -b参数表示创建并切换 或者 <===> git branch dev git checkout dev Creating a new branch is quick AND simple.
